anchors-away
============

Animate to Anchors

Extension to jQuery for animating to anchors on the DOM. 

Requirements
  jQuery

Usage
  $.anchorTo(someId);
  $(someId).anchorTo();

Other options:
  $.anchorTo(someId, durationMS);
  $.anchorTo(someId, durationMS, callbackFunction);
  $(someId).anchorTo(durationMS);
  $(someId).anchorTo(durationMS, callbackFunction);
